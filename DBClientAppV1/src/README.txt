# C195 Final Project

## Application Overview

**Purpose:** This application is a graphical user interface (GUI)-based scheduling desktop application designed for a global consulting organization. The organization operates internationally in multiple languages and has main offices in Phoenix, Arizona; White Plains, New York; Montreal, Canada; and London, England.

**Author:** Elham Pazhakh

**Contact:** Email - epazha2@wgu.edu

**Version:** Student Application Version 1

**Date:** October 18, 2023

## Development Environment

- **IDE:** IntelliJ IDEA 2023.2 (Community Edition)
- **JavaFX Version:** JavaFX-SDK-17.0.8
- **Java Development Kit (JDK):** 17.0.8
- **MySQL Connector Driver Version:** mysql-connector-java-8.0.25\mysql-connector-java-8.0.25.jar

## How to Run the Program

To run the program, follow these steps:

1. Open the project in IntelliJ IDEA.
2. Ensure that you have JDK 17.0.8 installed.
3. Configure your project to use Java 17.0.8.
4. Set up JavaFX in your project settings.
5. Build and run the application from IntelliJ IDEA.

## Additional Report (Part A3f)

**Description:** The additional report in Part A3f allows users to select a country and a division. It then displays the number of customers associated with that specific country and division. This feature provides valuable insights into the organization's customer distribution.
